Check Virtual Machine Encryption

The function checks whether a virtual machine is encrypted and if the encryption is properly configured. It uses the Azure Python SDK to retrieve the VM's encryption status.
Check Virtual Machine Acess

The function checks if access to a virtual machine is restricted and if the access policies are properly configured. It uses the Azure Python SDK to retrieve the public IP addresses associated with the VM's network interfaces and reports any public IP addresses found.
Check Multi-Factor Authentication Enabled

The function checks if multi-factor authentication (MFA) is enabled for the user account and if it is properly configured. It uses the Azure Python SDK to retrieve the user account and its associated MFA configuration.
Check Logging

The function checks if logging and monitoring are properly configured for the resources. It uses the Azure Python SDK to query for log data in Azure Monitor and reports whether any log data was found.
Check Patches

The function checks if the latest security patches are applied to the resources. It uses the Azure Python SDK to retrieve a list of all virtual machines in the subscription and checks each virtual machine for installed security patches.
